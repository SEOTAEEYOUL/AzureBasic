# Azure Basic

- 3-tier 기반의 Sample 을 Azure Cloud 에 구성하는 과정을 통해 익숙해지는 것을 목표로 함

## AS-IS
- Apache, Tomcat, MySQL/MariaDB 로 이루어진 서비스
- Apache : Static 문서 처리
- Tomcat : SpringFramework 기반 Biz 서비스 처리
